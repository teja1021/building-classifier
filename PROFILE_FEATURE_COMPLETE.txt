================================================================================
STATE MANAGEMENT
================================================================================

Local State (Profile.jsx):
  - formData: All editable profile fields
  - profileImage: Current profile image (base64 or URL)
  - tempImagePreview: Image preview before applying
  - isEditing: Toggle between view/edit modes
  - saveSuccess: Success message display
  - errors: Form validation errors
  - imageError: Image upload error messages

Global State (App.jsx):
  - user: Stored in React state and localStorage
  - User data persists across page refreshes
  - Updates automatically reflected across all pages

================================================================================
HOW TO USE
================================================================================

1. LOGIN
   - Sign in with your account (demo/demo123 for testing)

2. NAVIGATE TO PROFILE
   - Click "Profile" in the navigation menu
   - Or visit http://localhost:5173/profile

3. VIEW PROFILE
   - Your current profile information is displayed
   - Profile picture (or avatar with initials) shown on left

4. EDIT PROFILE
   - Click "Edit Profile" button
   - Update desired fields
   - Upload new profile image if needed
   - Click "Save Changes" to save

5. UPDATE IMAGE
   - Click "Upload Photo" button
   - Select image file (JPG, PNG, etc.)
   - Preview appears before applying
   - Click "Apply" to confirm or "Cancel" to discard

6. SAVE CHANGES
   - All fields validated before saving
   - Success message appears
   - Data saved to localStorage
   - Changes reflected across app
