ğŸ› COMMON ISSUES & FIXES
================================================================================

Issue: "Port 5173 already in use"
  Fix: npm run dev -- --port 3000
  (Uses port 3000 instead)

Issue: "Cannot find module"
  Fix: npm install
  (Reinstall dependencies)

Issue: "Backend not reachable"
  Fix: 
  1. Start backend: cd backend && uvicorn app.main:app --reload
  2. Check http://localhost:8000/ping in browser
  3. Restart frontend

Issue: "Changes not appearing (hot reload not working)"
  Fix:
  1. Stop dev server (Ctrl+C)
  2. npm install
  3. npm run dev

Issue: "Build fails"
  Fix:
  1. npm run build -- --debug (shows error details)
  2. Check error in console
  3. Fix code error
  4. Try again
